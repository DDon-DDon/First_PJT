# 테스트 케이스

## 테스트 케이스 ID 규칙

```
TC-{Feature}-{Number}

Feature:
- IN: 입고
- OUT: 출고
- REG: 제품 등록
- ADJ: 재고 조정
- DASH: 대시보드
- SYNC: 동기화

예시: TC-IN-001, TC-OUT-002
```

### TC-IN-001: 정상 입고 처리

| 항목 | 내용 |
|------|------|
| **사전 조건** | 제품이 시스템에 등록되어 있음, 현재 재고: 30개 |
